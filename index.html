Agentic ESG

Built by Beatriz Leal — Climate Risk & ESG Intelligence Platform

CrewAIClaude SonnetClaude Haiku FastAPIMongoDBRailway TCFDEN / PT / ES / FR / DE

AI-powered climate risk and ESG intelligence platform combining a hybrid Python deterministic engine with multi-agent orchestration and responsible AI design. Generates TCFD-aligned physical risk reports for any location worldwide.

Features a multilingual frontend (EN/PT/ES/FR/DE), historical comparison mode, batch CSV analysis, ESG glossary, Human-in-the-Loop validation, data quality protection, and PDF/Excel exports. Demo-validated with Brasília, Amsterdam, El Calafate, and Antarctica edge cases.

Agent pipeline: Climate Analyst → ESG Mapper (Sonnet, regulations KB) → Quality Judge (Sonnet) → Report Writer. Powered by 10 years of NASA satellite data + IPCC projections to 2050.

Agentic Support Platform

Built by Beatriz Leal — Capstone for "Become an Agentic Architect / Mastering Multiagent AI Systems with CrewAI"

May 2026CrewAI FlowFastAPI Claude Haiku 4.5Claude Sonnet 4.6Vanilla JS PostgreSQLRailway

Multi-agent AI customer support platform that combines CrewAI Flow orchestration, real-time external APIs, and LLM-as-judge quality evaluation to automate e-commerce support at scale.

A 9-step CrewAI Flow pipeline processes every ticket: parallel classification and sentiment analysis → deterministic routing with 13 priority levels → real-time API enrichment (ViaCEP + OpenWeatherMap + Refund DB) → knowledge RAG synthesis → LLM response generation → background quality evaluation by a stronger judge model.

Operators get a full dashboard with HITL approve/reject/await controls, per-agent token observability, cost forecasting (~$0.003272/ticket), and Excel + PDF export. Customers get personalized responses in ~8 seconds grounded in real operational data.
